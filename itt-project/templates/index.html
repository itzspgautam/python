About the Project

This project analyses the relationship between student sleep patterns and academic performance (CGPA). Using a real-world dataset of university students, we apply exploratory data analysis and a Linear Regression model to predict GPA based on daily habits such as study hours, sleep duration, physical activity, screen time, and sleep quality.
